# Code Assistant - Complete Project

A Python code assistant implementing all required features:
- ✅ System Prompt
- ✅ Memory Management  
- ✅ Vector Database (ChromaDB style)
- ✅ RAG Code Generation
- ✅ Router Layer
- ✅ Self-Learning

## Features
- Generate Python code
- Explain programming concepts
- Learn new code examples
- Memory of conversations
- Vector search for similar code

## Setup
1. Clone repository
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python main.py`

## Commands
- `generate [task]` - Generate code
- `/learn` - Teach new code
- `/memory` - Show conversation history
- `/db` - Show database stats
- `/exit` - Quit
